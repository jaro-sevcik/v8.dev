            +-------------------+

         +---+

      |  Non-rect |
      ++ +-----+ ++
       +-+     +-+

      +------------+
      |+-----------++     +------------+
      ||            |     |            |
      ||  CONTEXT   |<-+  |  CONTEXT   |<-+  +-----------+
      ++            |  |  |            |  |  | Even more |
       +------------+  |  +------------+  |  +-----------+
                       |                  |      ^
       +------------+  |  +------------+  |      |
       |    getY    |  |  |    getY    |  |      |
       |            |  |  |            |  |
       |     *------+--+  |     *------+--+
       |            |  |  |            |  |
       +------------+  |  +----+-------+  |  Standalone
                       |                  |
       +------------+  |  +----+-------+  |

             |                  |
             +----------+-------+
                        |
                        v
             +--------------------+
             |  SharedFunctionInfo|

             +----------+---------+
                        |
                        v

             |  unoptimized Code  |

      ??? <--+---* getX call      |
             |                    |
